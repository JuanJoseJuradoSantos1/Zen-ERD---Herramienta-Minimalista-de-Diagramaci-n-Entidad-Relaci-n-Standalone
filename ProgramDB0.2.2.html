    Ayuda de Controles - Zen ERD

    • Crear Entidad: Clic Derecho en lienzo vacío → "Add Entity Here"
    • Crear Atributo (Entidad/Relación): Clic Derecho sobre dueño → "Add Attr..."
    • Crear Relación: Arrastrar 🔵 de Entidad 1 → Soltar sobre Entidad 2 🟢
    • Seleccionar: Clic Izquierdo
    • Mover Entidad/Atributo: Clic Izq + Arrastrar (Entidad ajusta a cuadrícula si activa)
    • Mover Lienzo (Pan): Clic Izq + Arrastrar fondo
    • Zoom: Rueda / ➕ ➖ / + -
    • Editar: Doble Clic / Clic Derecho → "Edit..."
    • Redimensionar Entidad: Clic Der → "Resize..."
    • Redimensionar Rombo: Clic Der → "Resize Diamond" (Addon)
    • Eliminar: Seleccionar + Supr / Borrar O Clic Der → "Delete..."
    • Mostrar/Ocultar Barra: Mover ratón cerca del borde
    • Deshacer/Rehacer: ↩️ ↪️ / Ctrl+Z Ctrl+Y
    • Guardar/Cargar: 💾 📂
    • Exportar PNG: 🖼️
    • Activar/Desactivar Cuadrícula: ▦
    • Escalar UI: ➖ ➕ (junto al porcentaje)
    • Posición Barra: ⬆️ ⬇️ ⬅️ ➡️
    • Resetear Vista: 🎯 / 0
    • Configuración App: 🛠️ (Acceso a Addons, Ayuda, y más opciones)
    • Cambiar Tema: 🌓
    • Deseleccionar/Cancelar: Esc
